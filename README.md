# TabBurrow

タブを保存して管理するブラウザ拡張機能です。大量のタブを開いてしまうユーザーのために、メモリ使用量を削減し、作業効率を向上させます。

## ✨ 主な機能

- **タブの保存**: 開いているタブをワンクリックで保存し、ブラウザのタブリソースを解放します。
- **復元**: 保存したタブはいつでもクリック一つで復元できます。復元モード（通常/遅延読み込み/即時サスペンド）を選択可能。
- **タブグループ復元**: グループ内のタブをブラウザのネイティブタブグループとしてまとめて復元できます。
- **自動グループ化**: 保存したタブはドメインごとに自動的にグループ化され、見やすく整理されます。
- **カスタムグループ**: 独自のグループを作成し、プロジェクトや用途に合わせてタブを自由に整理できます。
- **検索機能**: タイトルやURLで保存済みのタブを瞬時に検索できます。検索オプション（大文字小文字区別/単語単位/正規表現）にも対応。
- **一括操作**: 複数のタブを選択して、一括削除やグループ移動が可能です。
- **自動収納**: 一定時間非アクティブなタブを自動的に収納することができます。細かいルール設定も可能。
- **リンクチェック**: 保存されたタブのURLが有効かどうかを一括でチェックし、リンク切れを検出できます。
- **インポート/エクスポート**: データのバックアップや他の環境への移行が可能です（JSON, URLリスト, Markdown対応）。
- **テーマ設定**: ライトモード/ダークモードの切り替えに対応しています。
- **自動バックアップ**: 設定した間隔でデータを自動的にバックアップし、世代管理を行います。
- **表示カスタマイズ**: グループ表示/フラット表示、通常/コンパクト表示を切り替え可能。デフォルトを設定画面で指定できます。
- **多言語対応**: 日本語と英語に対応しています。

## 🌟 TabBurrow のこだわり

快適なブラウジング体験のために、以下の点を重視して設計されています：

- **プライバシーファースト**: データはすべてブラウザ内部（IndexedDB）に保存されます。外部サーバーへの送信やアカウント登録は一切不要で、安心してご利用いただけます。
- **メモリ効率の最適化**: タブを単に非表示にするのではなく、完全に閉じてデータを保存することで、ブラウザのメモリ使用量を確実に削減します。
- **自由なデータ管理**: 保存したデータは JSON, Markdown, テキスト形式で自由に出力可能。バックアップやドキュメントへの活用も手軽に行えます。
- **ストレスフリーな整理**: ドメインごとの自動グループ化機能により、意識して整理しなくてもタブが自然にまとまります。
- **オープンソース**: 透明性の高い開発プロセスにより、信頼してご利用いただけるソフトウェアを目指しています。


## 📖 機能詳細

### 📥 タブの保存
ツールバーのアイコンをクリックすると、現在のウィンドウで開いているすべてのタブ（固定タブを除く）が保存されます。保存されたタブはブラウザから閉じられ、メモリを節約します。

また、ページ上で右クリックすると、コンテキストメニューから「カスタムグループに保存」を選択して、現在のタブを指定したグループに直接保存することもできます。

### 📂 グループ管理
保存されたタブは自動的に処理されます：
- **ドメイン自動グループ**: 同じドメインのタブは自動的にまとめられます（例: `github.com` のタブをまとめる）。
- **カスタムグループ**: 任意の名前でグループを作成し、関連するタブ（プロジェクト資料、旅行の計画など）をまとめて管理できます。

### 🔍 検索とフィルタリング
保存したタブが増えても心配ありません。上部の検索バーにキーワードを入力すると、タイトルやURLから即座に目的のタブを見つけ出せます。

- **グローバル検索**: 全タブを対象に検索
- **検索オプション**: 大文字小文字区別、単語単位マッチ、正規表現モードを切り替え可能
- **日時フィルタ**: 保存日時による絞り込み（今日、1週間、1ヶ月など）
- **ソート設定**: グループ順、タブ順をカスタマイズ可能

### ⏰ 自動収納
設定画面から自動収納を有効にすると、一定時間非アクティブなタブを自動的に収納します。

- 非アクティブ時間は30秒〜24時間の範囲で設定可能
- ドメインやタイトルによる除外ルールを設定可能
- 特定のタブを「保存のみ」「破棄」「固定タブ化」などのアクションに振り分け可能

### 🔗 リンクチェック
タブ管理画面から保存されたURLの有効性を一括チェックできます。

- リンク切れ（404など）を自動検出
- ステータスコード別のアクション設定
- 検出されたリンク切れタブを一括削除または移動

### 💾 インポート/エクスポート
設定画面の「データ管理」から、以下の形式でデータのバックアップや復元が可能です。
- **JSON形式**: 完全なデータのバックアップ用。
- **URLリスト**: シンプルなURLの羅列。テキストファイルとして保存できます。
- **Markdown**: タイトルとURLのリンク形式。ドキュメントへの貼り付けに便利です。

### 📦 自動バックアップ
設定画面から自動バックアップを有効にすると、定期的にデータを自動保存します。

- バックアップ間隔は1時間〜24時間のプリセット、またはカスタム間隔を設定可能
- 保持する世代数を設定し、古いバックアップを自動削除
- バックアップからの復元が可能

### ⚙️ アイコンクリック設定
ツールバーのアイコンをクリックした時の挙動をカスタマイズできます。

- **自動収納ルールの適用**: アイコンクリック時に自動収納のルールを適用するかどうか
- **固定タブの扱い**: 固定タブを「スキップ」または「サスペンド」に設定可能

## 📦 インストール

この拡張機能は Chrome と Firefox に対応しています。

- **Chrome ウェブストア**: [TabBurrow](https://chromewebstore.google.com/detail/tabburrow/cemnhbnnpblfbdamfdhjbkjccdefblpe)
- **Firefox Add-ons**: [TabBurrow](https://addons.mozilla.org/ja/firefox/addon/tabburrow/)

### 手動インストール

1. [Releases](https://github.com/Taremin/TabBurrow/releases) ページから最新のビルド済みパッケージをダウンロードし、解凍します。

#### Chrome / Edge
1. `chrome://extensions` を開きます。
2. 右上の「デベロッパーモード」をオンにします。
3. 「パッケージ化されていない拡張機能を読み込む」をクリックし、解凍したフォルダ内の `dist/chrome` を選択します。

#### Firefox
1. `about:debugging#/runtime/this-firefox` を開きます。
2. 「一時的なアドオンを読み込む...」をクリックし、解凍したフォルダ内の `dist/firefox/manifest.json` を選択します。

## 🛠️ 開発者向け情報

### 必要要件
- Node.js (v18以上推奨)
- npm

### セットアップ

リポジトリをクローンし、依存関係をインストールします。

```bash
git clone https://github.com/Taremin/TabBurrow.git
cd TabBurrow
npm install
```

### ビルド

**Chrome用ビルド:**
```bash
npm run build:chrome
# または
npm run build
```
`dist/chrome` ディレクトリに出力されます。

**Firefox用ビルド:**
```bash
npm run build:firefox
```
`dist/firefox` ディレクトリに出力されます。

**開発モード（ウォッチモード）:**
```bash
npm run watch          # Chrome
npm run watch:firefox  # Firefox
```

### テスト

**ユニットテスト:**
```bash
npm run test
```

**E2Eテスト:**
```bash
npm run e2e
```
