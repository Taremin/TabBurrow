# TabBurrow v1.3.1 リリースノート

## ✨ 新規機能・改善

- **タブカードプレビューの改善**: タブカードをホバーした際に表示されるスクリーンショットを、アスペクト比を維持したまま拡大表示するように改善しました。これにより、保存されているタブの内容が分かりやすくなりました。
- **設定画面のUI改善**: 設定画面にある「クレジット」リンクの視認性を向上させました。

## 🐛 バグ修正

- **コンテキストメニューの修正**: 
  - 拡張機能起動時に、コンテキストメニューのタイトルが言語設定を正しく反映しない問題を修正しました。
  - カスタムグループ関連の操作時におけるメニューの初期化エラー（`Cannot find menu item`）を修正しました。
- **UI表示の修正**: カスタムグループのメニュー等を表示する際のポップアップ位置計算ロジックを共通化・改善し、画面外にメニューがはみ出す（溢れる）問題を修正しました。

## 📝 ドキュメント

- **Android版Firefoxに関する対応**: Android版Firefox特有の制約事項や仕様に関する調査結果をドキュメント（`docs/firefox_android_limitations.md`）として追加しました。

---

*開発者向け: E2Eテストの安定性向上や未使用変数の削除など、内部の品質改善（リファクタリング、Lintエラーの解消等）を行いました。*
