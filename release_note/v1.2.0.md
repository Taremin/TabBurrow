# release/v1.2.0

## 概要
今回のアップデートでは、カスタムグループとドメイングループの管理機能が大幅に強化されました。また、大規模なリファクタリングにより安定性が向上し、新しい表示オプションや利便性の高い機能が多数追加されています。

## 新規機能
- **ドメイングループのピン留め**: よく使うドメイングループをリストの上部に固定できるようになりました。
- **ドラッグ＆ドロップによる順序変更**: カスタムグループの表示順序を直感的に並び替えられるようになりました。
- **カスタムカラー設定**: カスタムグループおよびピン留めされたドメイングループに任意の色を設定できるようになりました。
- **フル幅表示モード**: タブ管理画面の左右の余白をなくし、画面いっぱいにタブを表示できる設定を追加しました。
- **タブ管理画面のピン留め**: 拡張機能アイコンをクリックした際に、タブ管理画面をピン留めタブとして開く（または切り替える）設定を追加しました。
- **一時的なソート順変更**: タブ管理画面で、現在表示中のタブを一時的に別の順序（タイトル順、URL順、保存日時順など）で並び替えられる機能を追加しました。

## 改善・変更
- **カスタムグループ保存の挙動改善**: タブをカスタムグループに保存する際、既存の所属グループを維持したまま追加できるようになりました。
- **グループヘッダーの表示改善**: エイリアス（別名）を設定したドメイングループで、元のドメイン名も薄く表示されるようになり、判別しやすくなりました。
- **一括操作の強化**: 選択モードで「新規グループに一括移動（追加）」ができるようになりました。
- **ダイアログの改善**: タブ表示名やグループ名の変更ダイアログで、現在の名前が初期値として入力されるようになり、空入力によるリセットも可能になりました。
- **UIの最適化**: コンパクト表示時のレイアウト改善や、境界線の色調整により視認性が向上しました。
- **コード品質の向上**: 大規模なリファクタリングを実施し、メンテナンス性とパフォーマンスが向上しました。

## バグ修正
- 設定画面のヘッダーが正しく固定（sticky）されない問題を修正しました。
- 自動収納機能が Service Worker のサスペンド後に正しく再開されない問題を改善しました。
- コンパクト表示時にタブカードのホバーで 1px のちらつきが発生する問題を修正しました。
- タブの表示名（displayName）が特定の操作で消失する問題を修正しました。
- 特定の環境でダイアログのオーバーレイが正しく表示されない問題を修正しました。
- その他、細かなUI表示の崩れや不具合を多数修正しました。
